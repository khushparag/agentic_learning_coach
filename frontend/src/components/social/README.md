# Social Learning Components

This directory contains components for social learning features including peer challenges, solution sharing, study groups, and collaborative learning.

## Components

### Peer Challenges
- `PeerChallengesBrowser.tsx` - Browse and filter available challenges
- `ChallengeCard.tsx` - Individual challenge display card
- `CreateChallengeModal.tsx` - Modal for creating new challenges
- `ChallengeLeaderboard.tsx` - Challenge wins leaderboard

### Solution Sharing
- `SolutionSharingInterface.tsx` - Main solution sharing interface
- `SharedSolutionCard.tsx` - Individual shared solution display
- `SolutionComments.tsx` - Comments system for solutions
- `ShareSolutionModal.tsx` - Modal for sharing solutions

### Study Groups
- `StudyGroupsBrowser.tsx` - Browse and join study groups
- `StudyGroupCard.tsx` - Individual study group display
- `CreateStudyGroupModal.tsx` - Modal for creating study groups
- `StudyGroupProgress.tsx` - Group progress tracking

### Activity Feed
- `ActivityFeed.tsx` - Main activity feed component
- `ActivityFeedItem.tsx` - Individual activity item
- `FollowingList.tsx` - List of followed users

### Utilities
- `SocialEngagementMetrics.tsx` - Social engagement analytics
- `SocialNotifications.tsx` - Social notifications component

## Features

- **Peer Challenges**: Create and participate in coding challenges
- **Solution Sharing**: Share solutions with likes and comments
- **Study Groups**: Collaborative learning groups
- **Following System**: Follow other learners and see their activity
- **Real-time Updates**: Live updates for social interactions
- **Engagement Analytics**: Track social learning metrics
